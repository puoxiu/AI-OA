

## 创建项目
npm create vite@latest xingoa

然后按照提示操作：
* Project name: xingoa
* Select a framework: Vue
* Select a variant: JavaScript 或 TypeScript（推荐 TypeScript 但 JS 也可以）

运行
* npm install
* npm run dev

默认的项目结构(vite 默认模板)
```bash
xingoa/
├── index.html            # HTML 入口
├── src/
│   ├── assets/           # 静态资源
│   ├── App.vue           # 根组件
│   ├── main.js           # 应用入口
│   └── components/       # 组件目录
├── package.json

```
